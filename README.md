# DDNet Ghost C

This is a minimalistic C99-compatible ghost file library, inspired by the ghost file loading and saving code from [DDNet](https://github.com/ddnet/ddnet).

## Features

  * Loads ghost files.
  * Saves ghost files.
  * Simple and isolated.
  * The only dependency is libc.

## Usage

To use the ghost library in your project, simply include the `ghost_lib.h` file and compile the `ghost_lib.c` file along with your project.
